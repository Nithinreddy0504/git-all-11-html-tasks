Features of vs code :

  • Syntax Highlighting and Auto-Completion for most languages.
  • Integrated terminal for running scripts or commands.
  • Extensions marketplace to add functionality,such as language-specifc support,linters,debuggers,themes,and more.
  • Customizable interface with user-defined settingss and shortcuts.
