ELEMENTS OF HTML

There are three types of elements in html, there are as follows:

1. BLOCK LEVEL ELEMENT

2. INLINE-BLOCK ELEMENTS

3. INLINE ELEMENTS

1. BLOCK LEVEL ELEMENTS: the tag which occupies whole width and creates a new line
EX: h1-h6,P, div,semantic elements
2.INLINE-BLOCK ELEMENTS:the elements which occupy only require space and does not create a new line and we can apply width,height,margin andpadding properties.
EX:img, iframe,input, button.
3. INLINE ELEMENTS: the elements which occupies only required space and does not create a new line,in css we cannot apply width, height,padding top and bottom and margin top and bottom.
EX:sapn, formatting tags, b, strong, i
